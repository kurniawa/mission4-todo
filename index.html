        Terimakasih telah menggunakan Todo-App ini!

        Berikut adalah fitur-fitur yang terdapat pada Todo-App ini:

        1. Untuk fungsi penyimpanan data, Todo-App ini memanfaatkan localStorage.
        2. Anda dapat menentukan info user, yakni nama dan posisi/jabatan
        3. Anda dapat edit info user tersebut
        4. Pada saat insert todo, anda dapat menambahkan priority_level, detail/deskripsi todo anda, serta tanggal molai dan tenggat waktu dari todo yang ingin anda buat.
        5. Semua data (seperti: title, detail, priority_level, tanggal) dapat diedit/diubah apabila kurang sesuai
        6. Anda dapat mengubah status todo active menjadi done dan begitu juga sebaliknya.
        7. Anda dapat menghapus todo yang telah anda buat
        8. Anda dapat memanfaatkan select all atau select sebagian, lalu mengubah sekaligus dari active menjadi done atau sebaliknya, serta menghapus sekaligus semua todo yang telah ditandai.
